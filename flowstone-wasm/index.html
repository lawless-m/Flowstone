Flowstone WASM feasibility demo

A zip of Markdown notes goes in; a Datalog-queryable CozoDB comes out. No server-side processing — just this static page and a wasm blob. (FTS is disabled in the browser build: tantivy's segment-updater thread cannot spawn in a sandboxed wasm page, so search uses plain starts_with / substring Datalog instead.)

The default URL points at the CI-built zip on raw.githubusercontent.com, which sends Access-Control-Allow-Origin: *. Paste any URL from a host that does the same, or use the file picker below if CORS blocks you.
